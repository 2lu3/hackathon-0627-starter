## 参加手順

### 1. このリポジトリをForkする

画面右上の「Fork」ボタンをクリックしてください。

### 2. チームで開発を進める

ForkしたリポジトリをCloneして、チームで実装してください。

### 3. 提出物を揃える

以下の3ファイルを完成させてください。

- `src/UserRegistrationService.java` - リファクタリング後のコード
- `DESIGN.md` - 設計ドキュメント（テンプレートあり）
- `DECISIONS.md` - 設計判断ログ（テンプレートあり）

### 4. リポジトリURLを運営に共有する

発表前に、ForkしたリポジトリのURLをスタッフに共有してください。
